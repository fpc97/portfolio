About me

I’m a JavaScript developer and occasional designer. I’ve worked creating webfronts for small businesses as well as intuitive browser tools.

I specialize in React and modern frameworks such as Next and Gatsby. I’m also familiar with backend APIs and the Node.js ecosystem.

I have experience implementing full-fledged sites using SSG’s and SSR’s as well as with plain HTML, CSS, and JavaScript (such as this site!)
